.Stock Price Forecasting

This project focuses on time series forecasting of Microsoft stock closing prices using statistical and machine learning-based models.

.Objective
To analyze historical stock price trends and forecast future values using ARIMA and Facebook Prophet.

.Dataset
Historical Microsoft stock price data containing daily closing prices over multiple years.

.Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- Prophet
- Jupyter Notebook

.Key Analysis Steps
- Time series visualization
- Stationarity testing using ADF test
- Differencing to achieve stationarity
- ARIMA model fitting and evaluation
- Forecasting using Prophet
- Visualization of forecasts and components

.Key Insights
- The original time series was non-stationary and required differencing
- ARIMA captured short-term dependencies in the data
- Prophet modeled long-term trends and seasonality effectively
